Installing¶

Dependencies¶

Requirements

pywcsgrid2 is a python module that extends functionality of matplotlib, i.e., you need python with matplotlib installed. Note that you need at least 0.99.0 (or from svn) version of matplotlib which includes AxesGrid toolkit. The pyfits package is required to read fits files. Furthermore, you need either Kapteyn package or pywcs (from svn) installed.

matplotlib¶

  • Homepage
  • install from svn
  • 0.99.0rc1 : Check the mailing list or the home page for more recent version.
